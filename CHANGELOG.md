# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-27

### Added
- Initial project setup for GitHub Repo Finder.
- Core library for GitHub API search, ranking, and scoring.
- CLI interface for basic repository search.
- MCP server integration for LLM skill usage.
- Caching mechanism for token optimization.
- Basic test suite for core components.
- Initial documentation: README, SKILL.md, CONTRIBUTING.md.
